# auto-log README

该插件提供三项能力:

+ 在编辑器中选中的内容下一行添加 console.log('content', content)
    命令行操作: Insert Log
    快捷键:
        win: ctrl+w
        mac: cmd+shift+w
+ 删除当前文件中所有的console
    命令行操作: Del all Logs
    快捷键:
        win: ctrl+r
        mac: cmd+shift+r
+ 删除当前文件所有注释
    命令行操作: Remove all comments
    快捷键:
        win: ctrl+t
        mac: cmd+shift+t
